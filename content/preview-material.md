## Erstellungs-Guide

**Empfohlene Tools:**
- **Canva AI:** Thumbnails und Social-Media-Vorschauen
- **Excalidraw / Draw.io:** Workflow-Diagramme
- **OBS / Screen Studio:** Bildschirmaufnahmen für Preview-Videos
- **Screenshot-Tool:** Gnome Screenshot / Flameshot (Linux)

**Stilvorgaben:**
- Farbpalette: Indigo-Violett (#4F46E5), Türkis (#06B6D4), Dunkel (#0F172A)
- Schrift: Inter (Google Font)
- Thumbnail-Format: 1280x720px, PNG
- Video-Format: MP4, H.264, max 50MB
- Diagramme: Minimalistisch, weiße Linien auf dunklem Hintergrund

**Preview-Video-Skript (30s):**
```
[0-5s] Logo + Kurstitel einblenden
[5-15s] 3 Kern-Lernziele als Text-overlay (mit KI generiert)
[15-25s] 3 Screenshots aus dem Kurs (je 3s)
[25-30s] Call-to-Action: "Jetzt starten" + Link
```

## Automatisierte Erstellung mit KI

**Prompt für Canva AI / DALL-E / Midjourney:**
```
Erstelle ein Youtube-Thumbnail für einen Online-Kurs:
Titel: [KURSTITEL]
Hintergrund: Modernes Tech-Design, lila/blau Farbverlauf
Stil: Professionell, aber zugänglich, keine Menschen
Text: [KURSTITEL] (in großer, lesbarer Schrift)
Format: 1280x720px
```

**Prompt für Diagramme:**
```
Erstelle ein SVG-Diagramm eines [WORKFLOW-NAME]-Workflows.
Stil: Minimalistisch, dunkler Hintergrund,
helle Linien, Pfeile zwischen den Schritten,
rechteckige Boxen mit abgerundeten Ecken.
```
