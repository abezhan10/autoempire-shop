# Prompt Engineering Master Templates

## Prompt: Content-Brief Generator

**Anwendung:** Erstelle einen vollständigen Content-Brief für Texter oder KI.

```
Erstelle einen Content-Brief für folgendes Thema: [THEMA]
Zielgruppe: [ZIELGRUPPE]
Ziel: [INFORMEREN / VERKAUFEN / UNTERHALTEN]

Gib mir:
1. 5 Keyword-Ideen (SEO-primär)
2. 3 angle für den Artikel
3. Struktur (H2-Überschriften)
4. Call-to-Action-Vorschlag
5. Meta-Description (max 156 Zeichen)
6. 3 interne Verlinkungs-Vorschläge
```

## Prompt: Produkttext-Optimierer

**Anwendung:** Verbessere bestehende Produktbeschreibungen.

```
Optimiere folgenden Produkttext für höhere Conversion:

Aktueller Text:
[PRODUKTTEXT]

Verbessere:
1. Schreib den Text um — emotionaler, nutzenorientierter
2. Füge 3 Schmerzpunkte der Zielgruppe ein
3. Erstelle 3 Varianten der Überschrift
4. Füge Social-Proof-Elemente ein
5. Optimiere für SEO (primäres Keyword: [KEYWORD])
```

## Prompt: E-Mail-Sequenz-Generator

**Anwendung:** Erstelle eine automatisierte E-Mail-Sequenz.

```
Erstelle eine 5-teilige E-Mail-Sequenz für:

Produkt: [PRODUKTNAME]
Preis: [PREIS]
Zielgruppe: [ZIELGRUPPE]
Schmerzpunkt: [SCHMERZPUNKT]

Jede E-Mail:
- Betreffzeile
- Vorschau-Text
- Body (max 150 Wörter)
- Call-to-Action

Die Sequenz-Strategie:
E-Mail 1: Problem bewusst machen
E-Mail 2: Lösung vorstellen
E-Mail 3: Social Proof + Details
E-Mail 4: Dringlichkeit + Angebot
E-Mail 5: Letzte Chance + FAQ
```

## Prompt: Social-Media-Content-Plan

**Anwendung:** Erstelle einen monatlichen Content-Plan.

```
Erstelle einen Content-Plan für [MONAT] auf [PLATTFORM].

Brand: [NAME]
Ton: [FREUNDLICH/PROFESSIONELL/HUMORVOLL]
Ziele: [REICHWEITE / ENGAGEMENT / VERKÄUFE]
Themen: [3-5 THEMEN]

Pro Woche: 5 Posts
- 2 Educational
- 2 Entertaining
- 1 Promotional

Pro Post:
- Visual-Idee
- Caption (max 150 Zeichen)
- Hashtags (5)
- Beste Posting-Zeit

Bonus: 3 Story-Ideen pro Woche
```

## Prompt: Kunden-FAQ-Generator

**Anwendung:** Erstelle FAQ aus Kunden-E-Mails.

```
Erstelle eine FAQ aus folgenden Kundenanfragen:

[KUNDENANFRAGEN]

Strukturiere nach Kategorien:
1. Produkt/Service
2. Preis/Bezahlung
3. Technisches
4. Sonstiges

Pro FAQ-Eintrag:
- Frage (wie vom Kunden formuliert)
- Antwort (max 3 Sätze, freundlich)
- Related Questions (2-3)

Bonus: Erstelle 3 Chatbot-Antworten für häufige Fragen.
```

## Prompt: Meeting-Zusammenfassung

**Anwendung:** Fasse Meeting-Notizen strukturiert zusammen.

```
Fasse folgende Meeting-Notizen zusammen:

[NOTIZEN / TRANSKRIPT]

Format:
📋 Zusammenfassung (3 Sätze)
🎯 Entscheidungen (Liste)
✅ Action Items (mit Verantwortlichem)
📅 Nächste Schritte (mit Datum)
❓ Offene Fragen (wer muss klären)
```

## Prompt: SWOT-Analyse-Vorlage

**Anwendung:** Führe eine strukturierte SWOT-Analyse durch.

```
Führe eine detaillierte SWOT-Analyse durch für:

Thema: [PRODUKT / IDEE / UNTERNEHMEN]
Markt: [BRANCHE / MARKT]
Zeitraum: [12 MONATE]

Für jeden Bereich 5 Punkte:
Strengths (interne Stärken)
Weaknesses (interne Schwächen)
Opportunities (externe Chancen)
Threats (externe Risiken)

Abschließend:
- 3 strategische Empfehlungen
- 1 "Quick Win" (in 1 Woche umsetzbar)
- 1 "Big Bet" (hohes Risiko, hohe Belohnung)
```

[STELLENANZEIGE]

## Prompt: Projekt-Plan in 5 Minuten

**Anwendung:** Erstelle einen groben Projektplan.

```
Erstelle einen Projektplan für:

Projekt: [NAME]
Ziel: [ENDZIEL]
Deadline: [DATUM]
Team: [ANZAHL PERSONEN / ROLLEN]
Ressourcen: [BUDGET / TOOLS]

Gib mir:
1. Meilensteine (mit Datum)
2. Aufgaben pro Meilenstein
3. Kritisches Pfad (was ist abhängig?)
4. Risiken (Top-3)
5. Nächste 3 konkreten Schritte
```
