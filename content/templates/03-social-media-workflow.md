# Social Media Content Creation Workflow

## Überblick

Dieser Workflow führt dich Schritt für Schritt durch die Erstellung eines kompletten Social-Media-Content-Plans mit KI-Unterstützung.

**Ziel:** 30 Tage Content in 2 Stunden

---

| Frage | Deine Antwort |
|-------|---------------|
| Welche Plattform? | |

| Hauptziel (Reichweite/Engagement/Verkauf)? | |
| Ton (Freundlich/Professionell/Humorvoll)? | |
| 3 Hauptthemen? | |

### 1.2 KI-Strategie-Prompt

```
Erstelle eine Social-Media-Strategie für [PLATTFORM].

Zielgruppe: [ALTER, BERUF, INTERESSEN]
Themen: [3 THEMEN]
Ton: [TON]

Gib mir:
1. 5 Content-Säulen (Themen-Kategorien)
2. Beste Posting-Zeiten (laut Studien für diese Zielgruppe)
3. 3 Hashtag-Strategien
4. Content-Mix: Educational (40%) / Entertaining (35%) / Promotional (25%)
5. 3 Kampagnen-Ideen für den Monat
```

## Phase 3: Content-Produktion (45 Minuten)

### 3.1 Bild-Erstellung mit Canva AI

**Canva-Workflow:**
1. "Magic Design" — Beschreibe, was du brauchst
2. Wähle 3 Designs aus
3. Passe Farben an Branding an
4. Füge Text-Overlay hinzu
5. Exportiere als PNG/MP4

**Prompt für Canva AI:**
```
[THEMA] Social-Media-Post, modernes Design,
Farbpalette: Violett/Türkis/Gelb, minimalistisch,
kein Text auf dem Bild (nur im Overlay)
```

### 3.2 Caption-Schreiben mit KI

**Formel für jede Caption:**
```
1. Hook (erste 1-2 Zeilen) — Stop scrolling!
2. Problem nennen — "Kennst du das?"
3. Lösung zeigen — "Hier ist wie:"
4. Call-to-Action — "Speichern für später"
5. Hashtags (5 relevante)
```

**KI-Prompt:**
```
Schreibe eine Instagram-Caption.

Format: Bild mit [BESCHREIBUNG]
Thema: [THEMA]
Ton: [FREUNDLICH]

- Hook max 2 Zeilen
- 3 Bullet Points mit Mehrwert
- Call-to-Action
- 5 Hashtags

Option 1: Emotional
Option 2: Humorvoll
Option 3: Frage-basiert
```

## Checkliste: Tägliche Social-Media-Routine

- [ ] 10 Minuten: Kommentare beantworten (Priorität: Fragen > Lob > Kritik)
- [ ] 5 Minuten: 10 relevante Accounts interagieren (liken, kommentieren)
- [ ] 5 Minuten: Content für morgen prüfen (Bild + Caption)
- [ ] 2 Minuten: Stories posten (Hinter-den-Kulissen)

## Checkliste: Wöchentliche Routine

- [ ] Analyse der letzten Woche (30 Minuten)
- [ ] Content für nächste Woche vorbereiten (60 Minuten)
- [ ] 3 neue Content-Ideen sammeln (15 Minuten)
- [ ] Kommentar-Strategie anpassen (10 Minuten)
- [ ] Trending Topics/Hashtags checken (15 Minuten)

## Checkliste: Monatliche Routine

- [ ] Monats-Reporting erstellen (60 Minuten)
- [ ] Content-Kalender für nächsten Monat planen (90 Minuten)
- [ ] Strategie-Review: Was hat funktioniert? Was nicht? (30 Minuten)
- [ ] Neue Formate testen (Reels, Carousel, etc.) (45 Minuten)
- [ ] Wettbewerbs-Analyse (30 Minuten)
