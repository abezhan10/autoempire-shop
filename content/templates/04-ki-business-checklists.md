# KI Business Checklists

## Tägliche KI-Routine

### Morgen-Check (5 Minuten)
- [ ] KI-gestützte E-Mail-Priorisierung durchführen
- [ ] Tagesaufgaben von KI analysieren und priorisieren lassen
- [ ] Einen Prompt für die wichtigste Aufgabe des Tages schreiben
- [ ] KI-Erinnerungen für Deadlines prüfen

**Prompt für Tagespriorisierung:**
```
Analysiere diese To-Do-Liste und priorisiere nach:
1. Dringlichkeit (0-10)
2. Wichtigkeit (0-10)
3. Geschätzter Zeitaufwand
4. KI-Unterstützungs-Potenzial

To-Dos:
[TO-DO-LISTE]

Gib mir:
- Top-3 heute
- Was ich delegieren kann (an KI oder Team)
- Optimierte Reihenfolge
```

### Abend-Check (5 Minuten)
- [ ] KI-Zusammenfassung des Tages erstellen
- [ ] Nächste Schritte für morgen von KI vorbereiten lassen
- [ ] Erfolge und Learnings dokumentieren

**Prompt für Tagesreview:**
```
Fasse meinen Tag zusammen basierend auf:
- Erledigte Aufgaben: [LISTE]
- Nicht erledigt: [LISTE]
- Notizen: [NOTIZEN]

Gib mir:
1. 3 Dinge, die heute gut liefen
2. 1 Sache, die ich morgen anders machen sollte
3. Top-3-Prioritäten für morgen
```

## Wöchentliche KI-Checkliste

### Sonntag: Planung (30 Minuten)

**Content-Planung:**
- [ ] Letzte Woche analysieren (was hat funktioniert?)
- [ ] Content-Plan für kommende Woche mit KI erstellen
- [ ] KI-generierte Social-Media-Posts vorbereiten
- [ ] Blogartikel/Newsletter mit KI vorbereiten

**Business-Review:**
- [ ] KPIs der Woche von KI analysieren lassen
- [ ] Offene Aufgaben mit KI priorisieren
- [ ] Nächste Woche mit KI planen

**Prompt für Wochenplanung:**
```
Erstelle einen Wochenplan für [KALENDERWOCHE].
Letzte Woche: [KURZE ZUSAMMENFASSUNG]
Ausstehend: [OFFENE PUNKTE]

Gib mir:
- Montag: 3 Prioritäten
- Dienstag: 3 Prioritäten
- Mittwoch: 3 Prioritäten
- Donnerstag: 3 Prioritäten
- Freitag: 3 Prioritäten

Pro Tag: Welche Aufgabe hat KI-Potenzial?
```

---

## Monatliche KI-Checkliste (2 Stunden)

### Analyse & Optimierung

- [ ] Monats-KPIs von KI auswerten lassen
- [ ] KI-Nutzungs-Statistik prüfen (welche Prompts/Workflows genutzt?)
- [ ] Prompt-Bibliothek aufräumen und optimieren
- [ ] Neue KI-Tools testen (1 Tool pro Monat)
- [ ] Automatisierungs-Workflows auf Fehler prüfen
- [ ] 3 neue Automatisierungs-Ideen sammeln

**Prompt für Monatsanalyse:**
```
Führe eine Monatsanalyse durch für:

Daten:
- Umsatz: [ZAHL]
- Neue Kunden: [ZAHL]
- Content produziert: [ZAHL]
- Stunden gespart durch KI: [ZAHL]

Beantworte:
1. Top-3-Erfolge des Monats
2. Größte Zeitfresser
3. Welcher KI-Workflow hat am meisten gebracht?
4. Was sollte ich nächsten Monat anders machen?
5. 3 konkrete Ziele für nächsten Monat
```

## Checkliste: KI-Bereitschaft im Unternehmen

### Vor der Einführung
- [ ] Welche Aufgaben sind repetitiv und regelbasiert? (Beste KI-Kandidaten)
- [ ] Welche Daten sind verfügbar und sauber?
- [ ] Datenschutz-Anforderungen geklärt
- [ ] Budget für KI-Tools definiert
- [ ] Team-Skills eingeschätzt (Wer braucht Training?)
- [ ] Erfolgsmetriken definiert

### Während der Einführung
- [ ] Pilotprojekt mit 1-2 Workflows starten
- [ ] Ergebnisse dokumentieren (Vorher/Nachher)
- [ ] Team-Schulung durchführen
- [ ] Feedback-Routine etablieren
- [ ] Prompt-Bibliothek aufbauen
- [ ] Sicherheitsrichtlinien aktualisieren

### Nach der Einführung
- [ ] Regelmäßige KI-Audits (monatlich)
- [ ] Prompt-Qualität prüfen und verbessern
- [ ] Neue KI-Features evaluieren
- [ ] Kosten/Nutzen-Analyse
- [ ] Team-Erfahrungen teilen
- [ ] Nächste Automatisierungswelle planen

---

## Checkliste: KI-Content-Qualitätssicherung

### Vor Veröffentlichung
- [ ] Faktencheck: Sind alle Behauptungen korrekt?
- [ ] Bias-Check: Ist der Text ausgewogen?
- [ ] Tonfall: Passt der Text zur Marke?
- [ ] Zielgruppe: Ist die Sprache angemessen?
- [ ] SEO: Sind Keywords natürlich eingebaut?
- [ ] Call-to-Action: Ist klar, was der Leser tun soll?

### KI-Review-Prompt:
```
Prüfe folgenden Text auf:
1. Faktenfehler (was könnte falsch sein?)
2. Bias oder einseitige Darstellung
3. Tonfall-Konsistenz
4. Verständlichkeit für [ZIELGRUPPE]
5. SEO-Optimierung für [KEYWORD]

Text:
[CONTENT]

Gib eine Bewertung von 1-10 für jeden Punkt
und konkrete Verbesserungsvorschläge.
```

---

## Checkliste: KI-Workflow-Launch

### Vor dem Launch
- [ ] Workflow vollständig getestet
- [ ] Fehlerbehandlung implementiert
- [ ] Backup-Plan für Ausfälle
- [ ] Berechtigungen korrekt gesetzt
- [ ] Kostenkalkulation gemacht
- [ ] Dokumentation geschrieben

### Launch
- [ ] Workflow aktiviert
- [ ] Monitoring eingerichtet
- [ ] Team informiert
- [ ] Erste Ergebnisse dokumentieren
- [ ] Notfall-Kontakt hinterlegt

### Nach dem Launch (1 Woche)
- [ ] Ergebnisse mit Erwartungen vergleichen
- [ ] Team-Feedback einholen
- [ ] Optimierungen identifizieren
- [ ] Erfolgsmetriken aktualisieren
- [ ] Nächste Iteration planen

## Prompt: Eigene Checkliste erstellen

```
Erstelle eine Checkliste für [PROZESS].

Zielgruppe: [WER NUTZT SIE?]
Häufigkeit: [TÄGLICH/WÖCHENTLICH/MONATLICH]
Branche: [BRANCHE]

Die Checkliste soll:
1. Nicht mehr als 10 Punkte haben
2. Nach Priorität sortiert sein
3. Konkrete, überprüfbare Aktionen enthalten
4. Platz für Notizen lassen

Bonus: Welche 3 Punkte könnte man mit KI automatisieren?
```
